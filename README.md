# CSF_Lab03
#python lab programs

# Aim
To learn and implement basic Python programming concepts, including conditional statements, loops (for/while), and functions.

# List of Programs
1. q1_positive_negative.py: Check if a number is positive, negative, or zero.
2. q2_even_odd.py: Determine if a number is even or odd.
3. q3_largest_two.py: Find the larger of two numbers.
4. q4_largest_three.py: Find the largest of three numbers (Nested if-else).
5. q5_grades.py: Assign grades based on marks.
6. q6_menu_calculator.py: Simple menu-driven calculator.
7. q7_loop_1to10.py: Print 1 to 10 using a for loop.
8. q8_even_1to20.py: Print even numbers from 1 to 20.
9. q9_sum_10.py: Sum of first 10 natural numbers.
10. q10_table.py: Multiplication table of a given number.
11. q11_while_5.py: Print 1 to 5 using a while loop.
12. q12_sum_n.py: Sum of numbers from 1 to n.
13. q13_input_zero.py: Loop until user enters 0.
14. q14_break_at_6.py: Use 'break' to stop at 6.
15. q15_skip_5.py: Use 'continue' to skip number 5.
16. q16_search_list.py: Search for a target in a list.
17. q17_hello_func.py: Basic function definition.
18. q18_square_function.py: Function returning the square of a number.
19. q19_add_function.py: Function to add two numbers.
20. q20_func_even_odd.py: Function to check parity.
21. q21_factorial.py: Function to calculate factorial.
22. q22_table_function.py: Function for multiplication table.
23. q23_combined_even_odd.py: Combined loop and function task.
24. q24_final_calculator.py: Full integrated menu-driven calculator.
